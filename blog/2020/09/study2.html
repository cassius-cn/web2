Linux学习笔记

作者: Cassius

日期: 2020年9月2日

最基本最常用的20个命令，入门推荐

1. cat命令

cat fileName 显示文件内容（可以是多个文件）

2. ls命令

ls 显示目录下的内容 -l 显示详细的列表 -a显示隐藏文件

3.cp命令

cp file1 file2 将文件file1复制到file2（file1必须存在 file2不存在会新建）

cp file1 file2 file3 dir 将多个文件 复制到 目录dir中(file和dir必须存在)

4.mv命令

mv file1 file2 将file1重命名为file2

mv file1 file2 file3 dir 将多个文件移动到目录dir下

5.touch命令

touch file1 在当前命令下创建一个文件(如果文件已存在 只会改变时间戳 不会改变内容)

6.rm命令

rm file1 删除当前目录下的文件（不能删除目录）

7.echo命令

echo hello world 输出hello world

echo '*' 显示*

8.cd命令

cd dir 切换到目录dir中

cd .. 返回上级目录

9.mkdir命令

mkdir dir 在当前目录下 新建一个目录dir

10.rmdir命令

rmdir dir 移除当前目录下的dir目录（只能删除空目录）

rmdir -rf dir -r 删除所有子目录和子文件 -f 强制删除

11.shell通配符

*通配符 echo a* 查找当前目录下 所有a开头的文件 echo *a*所有包含a的文件

？通配符 echo te?t.txt 会匹配到 test.txt

12.grep命令

grep test /dir1/test1.txt 显示文件中包含文本参数test的所有行

grep test /dir1/* 显示dir目录下所有文件包含test的所有行

13.less命令

less file 分屏显示file文件的内容(空格键下一页 B键上一页 Q键退出)

14.pwd命令

pwd 显示当前目录的绝对路径

15.diff命令

diff file1 file2 显示该两个文件之间的不同

16.file命令

file file1 显示文件的格式信息

17.find和locate命令

find dir -name file -print 在目录dir中寻找file文件 locate用法一样（但可能查找不到新文件，因为新的索引还没建立）

18.head和tail命令

head file 显示文件前10行内容

head -6 file 显示文件前6行内容

tail file 显示文件最后10行内容

19.sort命令

sort file 将文件中内容按行开头字母顺序排序 显示出来(文件不变) -n 按数字顺序排序 -r反向排序

20.passwd命令

passwd 修改密码 第一遍旧密码 两边新密码
